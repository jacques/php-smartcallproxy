Provides an HTTP client using Guzzle around SmartCall's Restful (Webservices) Proxy.

## Installation

```
$ composer require jacques/php-smartcallproxy
```

```
{
    "require": {
        "jacques/php-smartcallproxy": "~0.1"
    }
}
```
